# CourseSelect 

此github项目为国科大研究生课程（高级软件工程）的课程作业，由李嘉禛和刘旭在原有选课系统样本的基础上进行更新改进完成，增添了部分功能，在完成改进的过程中，小组成员对Ruby on Rail这一web应用开发框架有了更好的认识。

### 原有功能功能：

* 多角色登陆（学生，老师，管理员）
* 学生动态选课，退课
* 老师动态增加，删除课程
* 老师对课程下的学生添加、修改成绩
* 权限控制：老师和学生只能看到自己相关课程信息

### 新增功能：

* 选课单功能
* 选课时间冲突检测功能
* 邮件通知功能
